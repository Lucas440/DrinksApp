# DrinksApp

This is a personal project app for android
The app is a Drinks ordering app where the user can input a drink with its price
They can then add a customer with an asigned seat
The app will let the user view what the customer has ordered already and add to that order tallying the total

This app is build for a phone and does not work well on a larger device such as a tablet as the proportions of the layout can cause problems
This something i will be looking into fixing
